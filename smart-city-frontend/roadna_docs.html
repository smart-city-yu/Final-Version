Token Check Logic (invisible)lines 41–54

After the delay, reads stored JWT → decodes exp claim → routes accordingly. No server call needed.

final token = await AuthService().getToken();
bool goHome = false;
if (token != null) {
  goHome = _isTokenValid(token);  // decode JWT exp locally
  if (!goHome) await AuthService().logout(); // expired → clear
}
Navigator.pushReplacementNamed(context, goHome ? '/home' : '/login');
Token is stored in SharedPreferences under key auth_token — see lib/services/auth_service.dart.
